# legacy/

여기는 현재 메인 실행 경로가 아닌 historical 파일을 모아둔 영역입니다.

## Included

- `docs/`
  - 과거 실험 메모/설계 문서
- `configs/`
  - 과거/백업 설정
- `inference/`
  - 과거 추론 스크립트
- `runners/`
  - 과거 experiment runner
- `run_scripts/`
  - 과거 shell launcher
- `train_variants/`
  - 과거 train variant

## Rule

- 현재 개발/운영 진입점으로 보지 않습니다.
- 필요할 때만 reference 용도로 읽습니다.
- 새 코드나 문서는 여기로 추가하지 않습니다.
